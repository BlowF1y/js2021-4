## [03월 23일]
논리 연산자
! = 부정 연산자
참을 부정으로 부정을 참으로 바꾼다
ex)
console.log(!true);      false
console.log(!false);      true
console.log(!(123>13))   false
console.log(!(123<12))   true

|| =  논리합 연산자
둘중 하나만 참이라도 참
ex)
true true = true
true false = true
false true = true
false false = false

&&   논리곱 연산자
둘중 하나만 false면 false
ex)
true true = true
true false = false
false true = false
false false = false
